# SQL-Queries
SQL QUERIES from Starting

> ***Create DataBase***


```js 
  create DATABASE users
```

> ***use Database***


```js 
    use users
```

> ***show all the tables***

```js
     show tables
```

> ***create table:~***

```js
     create table students(
        name varchar(255),
       email varchar(50),
      code varchar(10),
      batch varchar(30)
     );
```

> ***table content:~***

```js
    describe students
```

> ***Insert in to tables***

```js
     insert into students (name,email,code,batch) values("Anamika","a@gmail.com","pw1_006" ,"pt_web1") ;
```
